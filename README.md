
# Group 5 ETax-Invoice
Web application for create Electronic Tax-Invoice

## Prerequisites

* Spring Boot 1.5.10
* Java SDK (1.8 or upper version)
* IDE IntelliJ

## Built With

* Spring Boot Framework
* Java
* Maven

## Installation

> Clone this repository
> Build & Run on your IDE
